# WebDriver Demo

A WebDriver demo project.

## Usage

### Run tests

Run tests as follows:

```bash
./gradlew test
```

Specify `<browser>` argument `CHROME`, `EDGE` or `FIREFOX` and run tests as follows:

```bash
./gradlew test -Dbrowser=<browser>
```
